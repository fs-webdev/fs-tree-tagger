# birch-tree-tagger

An element that provides autocomplete suggestions for tagging to the tree


## Dependencies

Element dependencies are managed via [Bower](http://bower.io/). You can
install that via:

    npm install -g bower

Then, go ahead and download the element's dependencies:

    bower install
